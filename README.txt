== Description ==

Generate Amazon universal links to redirect users to the Amazon site of their country.

This plugin uses http://www.geoplugin.net to geolocate the user IP in order to redirect to the appropriate Amazon store.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `asin-smart-links` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
